# Web UI

`rarcane` does not ship a web UI. It is an upstream-client server whose supported
surfaces are MCP and CLI; Arcane remains the web interface and REST API.

The generic application/platform patterns in `docs/PATTERNS.md` describe how a
future scaffolded project may add API and web surfaces. Those optional patterns
are not part of this repository and commands referring to `apps/web` do not
apply to rarcane.
